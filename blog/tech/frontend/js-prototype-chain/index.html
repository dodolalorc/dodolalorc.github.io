      2. з»§жүҝдёҺжӣҙй•ҝзҡ„еҺҹеһӢй“ҫ

    JavaScript еҺҹеһӢй“ҫз®Җиҝ°

    2025-10-081,471 еӯ—4 еҲҶй’ҹйҳ…иҜ»#еүҚз«Ҝ#JavaScript#еҺҹеһӢй“ҫ
    еҲҶзұ»еүҚз«Ҝзі»еҲ—JavaScript еҹәзЎҖ 2/2

    зӣ®еҪ•

    1. еҺҹеһӢй“ҫ
      1. жһ„йҖ дёҖдёӘз®ҖеҚ•зҡ„еҺҹеһӢй“ҫдҫӢеӯҗ
      2. з»§жүҝдёҺжӣҙй•ҝзҡ„еҺҹеһӢй“ҫ
    2. еҺҹеһӢеҜ№иұЎ
      1. дёҖдёӘдҫӢеӯҗ
      2. еҸҜи§ҶеҢ–жӯЈзЎ®зҡ„з»“жһ„
      3. жӯЈзЎ®зҡ„зҗҶи§Јж–№ејҸ
    3. жҖ»з»“

    JavaScript жҳҜдёҖз§Қйқўеҗ‘еҜ№иұЎиҜӯиЁҖпјҢжҳҜдёҖз§ҚеҹәдәҺеҺҹеһӢзҡ„иҜӯиЁҖпјҢжҜҸдёӘеҜ№иұЎжӢҘжңүдёҖдёӘеҺҹеһӢеҜ№иұЎгҖӮ

    еҺҹеһӢй“ҫ

    еҺҹеһӢй“ҫВ  жҳҜ JavaScript дёӯе®һзҺ°з»§жүҝзҡ„жңәеҲ¶гҖӮжҜҸдёӘеҜ№иұЎйғҪжңүдёҖдёӘжҢҮеҗ‘еҸҰдёҖдёӘеҜ№иұЎзҡ„еҶ…йғЁй“ҫжҺҘпјҢиҝҷдёӘиў«жҢҮеҗ‘зҡ„еҜ№иұЎе°ұжҳҜе®ғзҡ„еҺҹеһӢгҖӮеҪ“иҜ•еӣҫи®ҝй—®дёҖдёӘеҜ№иұЎзҡ„еұһжҖ§ж—¶пјҢеҰӮжһңеңЁиҜҘеҜ№иұЎжң¬иә«жүҫдёҚеҲ°иҝҷдёӘеұһжҖ§пјҢйӮЈд№Ҳ JavaScript еј•ж“Һе°ұдјҡжІҝзқҖиҝҷдёӘй“ҫжҺҘеҲ°е®ғзҡ„еҺҹеһӢеҜ№иұЎдёҠеҺ»жүҫпјҢеҰӮжһңиҝҳжүҫдёҚеҲ°пјҢе°ұз»§з»ӯеҺ»еҺҹеһӢзҡ„еҺҹеһӢдёҠжүҫпјҢзӣҙеҲ°жүҫеҲ°иҜҘеұһжҖ§жҲ–иҖ…еҲ°иҫҫй“ҫжқЎзҡ„жң«з«ҜпјҲnullпјүдёәжӯўпјҢеҰӮжһңжүҫдёҚеҲ°иҝ”еӣһundefinedгҖӮиҝҷдёҖзі»еҲ—з”ұеҺҹеһӢеҜ№иұЎиҝһжҺҘиө·жқҘзҡ„й“ҫжқЎпјҢе°ұз§°дёәеҺҹеһӢй“ҫгҖӮ

    иҰҒзҗҶи§ЈеҺҹеһӢй“ҫпјҢйңҖиҰҒе…ҲдәҶи§ЈеҮ дёӘж ёеҝғжҰӮеҝөпјҡ

    • prototypeВ  еұһжҖ§пјҡ
      • иҝҷжҳҜеҮҪж•°зӢ¬жңүзҡ„еұһжҖ§гҖӮ
      • еҪ“дҪ дҪҝз”Ё В newВ  е…ій”®еӯ—и°ғз”ЁдёҖдёӘжһ„йҖ еҮҪж•°жқҘеҲӣе»әж–°еҜ№иұЎж—¶пјҢиҝҷдёӘж–°еҜ№иұЎзҡ„еҶ…йғЁеҺҹеһӢпјҲ[[Prototype]]пјүе°ұдјҡиў«и®ҫзҪ®дёәиҜҘжһ„йҖ еҮҪж•°зҡ„ В prototypeВ  еұһжҖ§жүҖжҢҮеҗ‘зҡ„еҜ№иұЎгҖӮ
      • з®ҖеҚ•иҜҙпјҡеҮҪж•°зҡ„ В prototypeВ  еұһжҖ§пјҢжҳҜдёәе°ҶжқҘйҖҡиҝҮе®ғеҲӣе»әзҡ„е®һдҫӢеҜ№иұЎеҮҶеӨҮзҡ„вҖңи“қеӣҫвҖқжҲ–вҖңжЁЎжқҝвҖқгҖӮ
    • [[Prototype]]В  еҶ…йғЁеұһжҖ§пјҡ
      • иҝҷжҳҜжҜҸдёӘеҜ№иұЎйғҪжңүзҡ„дёҖдёӘеҶ…йғЁеұһжҖ§пјҢжҢҮеҗ‘е®ғзҡ„еҺҹеһӢеҜ№иұЎгҖӮ
      • еңЁд»Јз ҒдёӯпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮ В Object.getPrototypeOf(obj)В  ж–№жі•жқҘиҺ·еҸ–е®ғпјҢжҲ–иҖ…йҖҡиҝҮйқһж ҮеҮҶдҪҶиў«и®ёеӨҡжөҸи§ҲеҷЁе®һзҺ°зҡ„ В obj.__proto__В  еұһжҖ§жқҘи®ҝй—®гҖӮ
    • constructorВ  еұһжҖ§пјҡ
      • й»ҳи®Өжғ…еҶөдёӢпјҢеҮҪж•°зҡ„ В prototypeВ  еұһжҖ§жҳҜдёҖдёӘеҜ№иұЎпјҢиҝҷдёӘеҜ№иұЎжңүдёҖдёӘ В constructorВ  еұһжҖ§пјҢжҢҮеӣһиҜҘеҮҪж•°жң¬иә«гҖӮ

    жһ„йҖ дёҖдёӘз®ҖеҚ•зҡ„еҺҹеһӢй“ҫдҫӢеӯҗ

    жӯӨеҺҹеһӢй“ҫзҡ„е…ізі»жҳҜпјҡ

    • person1В  жң¬иә«жңү В nameВ  еұһжҖ§гҖӮ
    • еҪ“и°ғз”Ё В person1.sayHello()В  ж—¶пјҢJS еј•ж“Һпјҡ
      1. е…ҲеңЁ В person1В  еҜ№иұЎиҮӘиә«дёҠжүҫ В sayHelloВ  ж–№жі•пјҢжІЎжүҫеҲ°гҖӮ
      2. з„¶еҗҺйҖҡиҝҮ В person1.__proto__пјҲд№ҹе°ұжҳҜ В Person.prototypeпјүеҺ»жүҫпјҢжүҫеҲ°дәҶпјҒдәҺжҳҜжү§иЎҢгҖӮ
    • Person.prototypeВ  д№ҹжҳҜдёҖдёӘеҜ№иұЎпјҢе®ғзҡ„еҺҹеһӢжҳҜ В Object.prototypeпјҲиҝҷжҳҜжүҖжңүжҷ®йҖҡеҜ№иұЎзҡ„й»ҳи®ӨеҺҹеһӢпјүгҖӮ
    • Object.prototypeВ  зҡ„еҺҹеһӢжҳҜ В nullпјҢиЎЁзӨәеҺҹеһӢй“ҫзҡ„з»ҲзӮ№гҖӮ

    з»§жүҝдёҺжӣҙй•ҝзҡ„еҺҹеһӢй“ҫ

    жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮи®©дёҖдёӘжһ„йҖ еҮҪж•°зҡ„ В prototypeВ  жҢҮеҗ‘еҸҰдёҖдёӘжһ„йҖ еҮҪж•°зҡ„е®һдҫӢпјҢжқҘе®һзҺ°з»§жүҝпјҢд»ҺиҖҢе»¶й•ҝеҺҹеһӢй“ҫгҖӮ

    е’ҢдёҠйқўдёҖж®ө Person зҡ„д»Јз ҒдҫӢеӯҗдёҖиө·зңӢ

    жӯӨж—¶зҡ„еҺҹеһӢй“ҫе…ізі»жҳҜпјҡ

    • еҪ“и°ғз”Ё В student1.sayHello()В  ж—¶пјҢJS еј•ж“Һпјҡ
      1. еңЁ В student1В  дёҠжүҫпјҢжІЎжүҫеҲ°гҖӮ
      2. еңЁ В Student.prototypeВ  дёҠжүҫпјҢжІЎжүҫеҲ°гҖӮ
      3. еңЁ В Person.prototypeВ  дёҠжүҫпјҢжүҫеҲ°дәҶпјҒжү§иЎҢгҖӮ

    еҺҹеһӢеҜ№иұЎ

    дёҖдёӘеҜ№иұЎзҡ„еұһжҖ§е’Ңж–№жі•е®ҡд№үеңЁ Object зҡ„жһ„йҖ еҷЁеҮҪж•°пјҲconstructorпјүдёҠзҡ„prototypeеұһжҖ§дёҠпјҢиҖҢйқһе®һдҫӢеҜ№иұЎжң¬иә«гҖӮ

    еҪ“жҲ‘д»¬е®ҡд№үдёҖдёӘеҮҪж•°еҜ№иұЎж—¶пјҢжҜҸдёӘеҮҪж•°йғҪжңүдёҖдёӘзү№ж®Ҡзҡ„еұһжҖ§пјҢеҸ«еҺҹеһӢprototypeгҖӮ

    еҸҜд»ҘеңЁжҺ§еҲ¶еҸ°жҹҘзңӢиҝҷдёӘеұһжҖ§зҡ„еҶ…е®№пјҡ

    иҫ“еҮәзҡ„иҝҷдёӘе°ұжҳҜеёёиҜҙзҡ„еҺҹеһӢеҜ№иұЎгҖӮ

    е…¶дёӯзҡ„constructorжҳҜеҺҹеһӢеҜ№иұЎзҡ„дёҖдёӘиҮӘжңүеұһжҖ§пјҢиҝҷдёӘеұһжҖ§жҢҮеҗ‘иҜҘеҮҪж•°гҖӮ

    е’Ңconstructorе№¶еҲ—зҡ„жҳҜ[[Prototype]]пјҢиҝҷжҳҜдёҖдёӘеҶ…йғЁй“ҫжҺҘпјҢжүҖжңүеҜ№иұЎйғҪжңүиҝҷдёӘеұһжҖ§пјҢиҝҷдёӘеұһжҖ§жҢҮеҗ‘е®ғзҡ„еҺҹеһӢгҖӮ

    __proto__еұһжҖ§жҳҜеҜ№иұЎе®һдҫӢе’Ңе®ғзҡ„жһ„йҖ еҷЁзҡ„й“ҫжҺҘпјҢжҳҜд»Һжһ„йҖ еҮҪж•°зҡ„prototypeеұһжҖ§жҙҫз”ҹзҡ„пјҢд»ҺиҝҷдёӘй“ҫжҺҘеҗ‘дёҠжәҜжәҗеҺҹеһӢй“ҫпјҢеҸҜд»ҘеңЁжһ„йҖ еҷЁеңЁжүҫеҲ°иҝҷдәӣеұһжҖ§е’Ңж–№жі•гҖӮ

    дёҖдёӘдҫӢеӯҗ

    е…ій”®зӮ№пјҡеҺҹеһӢй“ҫжҳҜйҖҡиҝҮ В [[Prototype]]пјҲеҚі В __proto__пјүй“ҫжҺҘзҡ„пјҢиҖҢдёҚжҳҜйҖҡиҝҮ В prototypeВ  еұһжҖ§гҖӮ

    • Person.prototypeВ  жҳҜдёҖдёӘжҷ®йҖҡеҜ№иұЎ
    • жҷ®йҖҡеҜ№иұЎжІЎжңү В prototypeВ  еұһжҖ§пјҢеҸӘжңү В [[Prototype]]В  й“ҫжҺҘ
    • жңҖз»ҲпјҢObject.prototype.__proto__В  жҢҮеҗ‘ В nullпјҢеҺҹеһӢй“ҫз»Ҳжӯў

    еҸҜи§ҶеҢ–жӯЈзЎ®зҡ„з»“жһ„

    жӯЈзЎ®зҡ„зҗҶи§Јж–№ејҸ

    еҺҹеһӢй“ҫзҡ„й“ҫжҺҘжҳҜйҖҡиҝҮ В [[Prototype]]пјҲеҚі В __proto__пјүе»әз«Ӣзҡ„пјҢеҪўжҲҗиҝҷж ·зҡ„й“ҫжқЎпјҡ

    жҖ»з»“

    жҰӮеҝөиҜҙжҳҺи®ҝй—®ж–№ејҸ
    еҺҹеһӢй“ҫеҜ№иұЎд№Ӣй—ҙйҖҡиҝҮеҺҹеһӢиҝһжҺҘиө·жқҘзҡ„з»§жүҝй“ҫжқЎгҖӮ-
    prototypeеҮҪж•°зҡ„еұһжҖ§пјҢжҢҮеҗ‘дёҖдёӘеҜ№иұЎпјҢиҜҘеҜ№иұЎе°ҶжҲҗдёәжңӘжқҘе®һдҫӢзҡ„еҺҹеһӢгҖӮFunc.prototype
    [[Prototype]]еҜ№иұЎзҡ„еҶ…йғЁеұһжҖ§пјҢжҢҮеҗ‘е®ғзҡ„еҺҹеһӢгҖӮObject.getPrototypeOf(obj)В  жҲ– В obj.__proto__
    constructorеҺҹеһӢеҜ№иұЎдёҠзҡ„еұһжҖ§пјҢжҢҮеӣһеҲӣе»әе®ғзҡ„жһ„йҖ еҮҪж•°гҖӮobj.constructor
    JavaScript еҹәзЎҖ дёҠдёҖзҜҮJavaScript й“ҫејҸи°ғз”Ёи®ҫи®ЎжЁЎејҸ | VueдёҫдҫӢ
