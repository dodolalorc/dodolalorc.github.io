Codeforces Round 980 (Div. 2)#

#A. Profitable Interest Rate

预计阅读时间: 5 分钟

#题意

有两种储值方式——无利可图和盈利，“盈利”可以保证盈利，但是有最低储值要求，“无利可图”类型没有利息，但是可以让“盈利”的最低储值降低。在“无利可图”储值xxx元，可以让“盈利”的最低值要求降低2×x2\times x2×x元，最低储值不能低于000元，两种储值均不能取出。现在Alice拥有aaa元，并想使存入“盈利”的金额越多越好，求Alice最多存入多少“盈利”类型的金额。

#数据范围

  • 1≤t≤1041\leq t\leq 10^41≤t≤104
  • 1≤a,b≤1091\leq a,b\leq 10^91≤a,b≤109

#思路

假设存入xxx元在“无利可图”，则“盈利”的最低储值降为b−2×xb-2\times xb−2×x，此时如果Alice的剩余金额a−xa-xa−x可以达到最低储值，则答案为a−xa-xa−x，否则为000，最小化xxx即可。

#参考代码

void solve() {
  ll a, b;
  cin >> a >> b;
  ll x = max(0ll, b - a);
  ll ans = max(0ll, a - x);
  cout << ans << '\n';
}

#B. Buying Lemonade

#题意

柠檬水售卖机有nnn个按钮，但是无法辨认每个按钮对应的槽位还剩多少柠檬水，购买者知道每个槽位最初的柠檬水数量，每按下一个按钮，如果该按钮对应的槽位还有柠檬水，则可以售出111瓶柠檬水，若槽位为空，则将没有任何东西掉出来。现在需要精确购买kkk瓶柠檬水，保证柠檬水售卖机里有足够的柠檬水，即∑i=1nai≥k\sum_{i=1}^{n} a_i \ge k∑i=1n​ai​≥k。求问最少的可以保证达成任务的点击次数。

#数据范围

  • 1≤t≤1041\leq t\leq 10^41≤t≤104
  • 1≤n≤2×1051\leq n\leq 2\times 10^51≤n≤2×105
  • 1≤k≤1091\leq k\leq 10^91≤k≤109
  • 1≤ai≤1091\leq a_i\leq 10^91≤ai​≤109

nnn之和不超过2×1052\times 10^52×105。

#思路

基本的贪心购买思路是一直按同一个按钮，直到该槽位为空，考虑到购买kkk瓶至少需要kkk次基本点击，我们需要最小化会浪费的点击次数，即点空槽位的次数。

假设当前每个槽位都至少有222瓶柠檬水，我们想最小化点空的次数，最好先把每个按钮都点击222次，如果此时的总数足够kkk，我们就计数次数，否则去掉空槽位（通过 1 次点击去除），我们要继续最小化。

二分在每个按钮点击的次数，查询能买到的柠檬水总数，计数最少的次数，即第一个满足总数大于等于kkk的那个次数。

#参考代码

ll a[maxn];

void solve() {
  ll n, k;
  cin >> n >> k;
  for (ll i = 1; i <= n; i++) {
    cin >> a[i];
  }
  sort(a + 1, a + 1 + n);

  auto get_cnt = [&](ll cnt) {
    ll ret = 0ll, tot = 0ll;
    for (ll i = 1; i <= n; i++) {
      if (a[i] < cnt) {
        if (tot + a[i] < k) {
          ret += a[i] + 1;
          tot += a[i];
        } else {
          ret += k - tot;
          break;
        }
      } else {
        if (tot + cnt < k) {
          ret += cnt;
          tot += cnt;
        } else {
          ret += k - tot;
          break;
        }
      }
    }
    return ret;
  };
  auto get_lemonade = [&](ll cnt) {
    ll tot = 0ll;
    for (ll i = 1; i <= n; i++) {
      if (a[i] < cnt) {
        tot += a[i];
      } else {
        tot += cnt;
      }
    }
    return tot;
  };

  // 可以获得k的第一个次数的下标
  ll l = 1, r = n;
  while (l < r) {
    ll mid = l + (r - l) / 2;
    if (get_lemonade(a[mid]) < k) {
      l = mid + 1;
    } else {
      r = mid;
    }
  }
  ll ans = get_cnt(a[l]);
  cout << ans << '\n';
}

#C. Concatenation of Arrays

#题意

有nnn个二元组，将这些二元组重新排序后让序列的逆序数最小，输出排序后的结果。

#数据范围

  • 1≤t≤1041\leq t\leq 10^41≤t≤104
  • 1≤n≤1051\leq n\leq 10^51≤n≤105
  • 1≤ai,j≤1091\leq a_{i,j}\leq 10^91≤ai,j​≤109

#思路

排序，将二元组按照两数之和、第一位数、第二位数的优先级顺序从小到大排序。

#参考代码

void solve() {
  ll n;
  cin >> n;
  vector<pll> v;
  for (ll i = 1; i <= n; i++) {
    pll p;
    cin >> p.first >> p.second;
    v.push_back(p);
  }
  sort(v.begin(), v.end(), [&](pll p1, pll p2) {
    return make_tuple(p1.first + p1.second, p1.first, p1.second) <
           make_tuple(p2.first + p2.second, p2.first, p2.second);
  });
  for (auto i : v) {
    cout << i.first << ' ' << i.second << ' ';
  }
  cout << '\n';
}

#D. Skipping

#题意

问答系统，对第iii个题，如果选择回答，可以获得aia_iai​分数，接下来只能选择序号j<ij\lt ij<i且没有操作过的问题。也可以选择跳过，接下来可以选择j≤bij\leq b_ij≤bi​的未操作过的题目。从111号问题开始回答。

#思路

注意到，当跳到索引iii时，所有1≤k≤i1\leq k\leq i1≤k≤i的没有回答过的题目，都可以通过依次回答。我们只需要贪心的选择前置和-到达某个位置的最小代价即可。

考虑用一个全局multisetmultisetmultiset维护当前可以使用的代价，从111号问题开始，每个问题的bib_ibi​指向一个到bi+1b_i+1bi​+1会“失效”的最小代价，每次移动时去除失效代价，并加入新的最小代价，维护最大得分即可。

维护时可以注意到bi≤ib_i\leq ibi​≤i的跳题没有意义，不如直接向前答题，可以通过判断去除这种移动。

#参考代码

void solve() {
  cin >> n;
  for (ll i = 1; i <= n; i++) {
    cin >> a[i];
    pre[i] = pre[i - 1] + a[i];
  }
  for (ll i = 1; i <= n; i++) {
    cin >> b[i];
  }

  // ll ans = a[1];
  multiset<ll> st;        // 可行代价
  map<ll, vector<ll>> mp; // 存储到达i的所有可行代价
  mp[2].push_back(0);
  st.insert(0);
  ll ans = a[1];
  for (ll i = 1; i <= n; i++) {
    for (auto x : mp[i]) {
      st.erase(st.find(x)); // 删去不再适用的代价
    }

    if (st.empty()) {
      dis[i] = -1; // 没有可用的代价
    } else {
      dis[i] = *st.begin();
    }

    if (dis[i] == -1)
      continue;
    ans = max(ans, pre[i] - dis[i]);

    if (b[i] <= i)
      continue;
    // 可以拓展到点b[i],b[i]+1时刻这些代价都不再适用
    ll d = dis[i] + a[i];
    mp[b[i] + 1].push_back(d);
    st.insert(d);
  }

  cout << ans << '\n';
}
